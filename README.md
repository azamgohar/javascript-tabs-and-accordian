## 🔽 Accordion (FAQ)

The accordion allows users to click on a question and reveal the answer.

### Key Concept:

* Clicking a question toggles the `.active` class on its parent
* CSS handles the show/hide using height or max-height

---

## 📑 Tabs

Tabs allow switching between different content sections without reloading the page.

### Key Concept:

* Clicking a tab activates it
* Only one tab content is visible at a time
* Uses `.active` class for switching
